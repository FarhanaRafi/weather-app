# Weather App

A simple weather app built with React and TypeScript.

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and go to `http://localhost:5173`

## Features

- Search for cities
- View current weather
- 7-day forecast
- Responsive design

## Technologies Used

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Open-Meteo API
